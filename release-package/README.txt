========================================
  SESLİ GÜNLÜK PLANLAYICI v1.0.1
========================================

KURULUM:
--------
1. ZIP dosyasını bir klasöre çıkartın
2. "Sesli Günlük Planlayıcı 1.0.0.exe" dosyasını çalıştırın
   VEYA
   "Sesli_Gunluk_Planlayici_Calistir.bat" kısayolunu kullanın

İLK KULLANIM:
-------------
1. Ayarlar bölümüne gidin
2. Google Gemini API anahtarınızı girin
   (API anahtarı için: https://aistudio.google.com/app/apikey)
3. Kaydet'e basın
4. Artık yapay zeka ve sesli tanıma özelliklerini kullanabilirsiniz!

ÖZELLİKLER:
-----------
✅ Sesli not alma ve görev ekleme
✅ Yapay zeka destekli görev yönetimi
✅ Günlük not defteri
✅ Görev kategorilendirme ve analiz
✅ Arşiv sistemi
✅ Karanlık/Aydınlık tema
✅ Bildirim ve hatırlatıcılar

SORUN GİDERME:
--------------
- Sesli tanıma çalışmıyorsa: API anahtarınızı kontrol edin
- Veriler kayboluyorsa: Tarayıcı önbelleğini temizlemeyin
- Uygulama açılmıyorsa: Windows Defender'ı kontrol edin
