🚀 HIZLI BAŞLANGIÇ
═══════════════════════════════════════════════════════════════

ADIM 1: Uygulamayı Kur
─────────────────────────────────────────────────────────────
• Setup dosyasını çalıştır
• Kurulum sihirbazını takip et
• Masaüstü kısayolu oluştur

ADIM 2: API Anahtarı Al
─────────────────────────────────────────────────────────────
• https://makersuite.google.com/app/apikey adresine git
• Google hesabınla giriş yap
• API anahtarını kopyala

ADIM 3: Uygulamayı Yapılandır
─────────────────────────────────────────────────────────────
• Uygulamayı aç
• Profil → Ayarlar'a git
• API anahtarını yapıştır
• Kaydet

ADIM 4: İlk Görevini Ekle
─────────────────────────────────────────────────────────────
• Mikrofon butonuna tıkla
• "Yarın saat 10'da toplantı" de
• VEYA "+" butonuyla manuel ekle
